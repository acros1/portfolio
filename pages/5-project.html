Innovative Project

Supervized by Vinvent Migliore
The innovative project covers all the Innovative Smart System semester. It is a professional collaborative project with a company in order to apply every knowledge learned during modules.

Our team is comprised of six engineering students, three Computer Science and Network engineering students and three Electronics and Automation students. The Smart Cabin project was chosen among a list of available projects for its industrial relevancy and the opportunity to work in cooperation with STERELA and Airbus.

Our client is STERELA, an Airbus subcontractor. They are working on a proof of concept to demonstrate to Airbus the feasibility of a project in order to obtain funds from Airbus to advance to the production stage.

Airbus currently tests each plane on the ground before its initial flight. To do this, they need to connect different wires from a computer to different sensors. In this way, the computer can send simulated data in order to retrieve sensors responses which allow to check the correct operating of the plane. The issue is that a cabin can be 70m long and the tests can have thou-sands of different configurations which require immense cable lengths, workforce and time to set up. The purpose of this project is to implement a wireless interface (gateway) between the plane and the computer (called the ESAO workstation corresponding to "Essais au Sol Assist√©s par Ordinateur") to manage tests for them to communicate. The gateway has to detect the planes configuration, send test signals to the plane through a radio module and receive the returned values. In addition to this "translation work", the gateway must be an autonomous system in order to limit the time operators have to spend on it - for instance, it needs to update automatically.

STERELA is implementing an automated testing tool for Airbus which is composed of two parts: a test box located on the plane which implements the tests, and a gateway to mediate wireless communication between the test box and a workstation. Making this communication wireless has many advantages such as making the testing easier, cheaper and faster as no cables are needed.

This project represents a complete professional experience. Working for clients as STERELA and Airbus with imposed specifications is a real challenge. We encountered a lot of difficulties and technical challenges, but it was very rewarding to work as team and progress together. Furthermore, regarding the complex sanitary situation due to Covid-19 it was very enjoyable to be enrolled in a project of that scale. We also had the chance to visit a part of the Airbus campus based at Saint-Martin-du-Touch which the biggest European industrial site. This experience is certainly a real asset in order to fine-tune my profile for the labour market.
