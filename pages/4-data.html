Analysis and Data processing - Business applications

Supervized by Nawal Guermouche
In the area of IoT expansion in the internet world the ability to deal with a big quantity of data become inescapable. This course gathers skills about different methods in order to manage perfectly the enormous data amount retrieved by IoT systems.
  • Desriptive part
  • Technical part
  • Analytic part

This course is composed of two part and brings skills about data management. The first part gives competency about ontology and data manipulation that a reasoner permit. The second part is an introduction to R language and big data.

Semantic web of thing

The semantic web of thing introduction is composed of practical sessions in which we use Protégé in order to create an ontology and understand what the reasoner can deduce. In this case we create a smart meteorology application by describing weather stations in order to makes their data strongest as possible.

Processing and Analysis of Data

The concrete part of this course is a project in which we have to analyse a dataset using R language. We choose a dataset including homicides information from 1980 to 2014 and we tried to extract as much information as we could.

This module introduces to an interesting notion which surround the IoT environment. The big amount of data is basically the aim of an IoT system which try to retrieve as much information as it could. However, one asset of an effective IoT system is defined by the efficiency of the data treatment.
It is useful to gain competency on big data and be able to have a critical eye on a dataset for instance. It represents the best way to reach the best performance while designing an IoT system.
