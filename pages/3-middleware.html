Middleware and Service

Supervized by Thierry Monteil
As one of the biggest supports for IoT is Internet this module brings acquaintances rules and technologies in order to understand how every IoT devices communicates and act together.
  • Desriptive part
  • Technical part
  • Analytic part

This module integrates three parts and gives an overview of IoT services architecture and possibilities.
The aim of the first part is to understand the service-oriented architecture, web services and underlying issues of distributed applications.
The second part gives and overview of many IoT standards and highlight OneM2M.
The last part brings acknowledgments on virtualization, cloud computing, autonomic computing and edge computing.

Service oriented architecture

The concrete application of this part is a project based on INSAâ€™s rooms automated management. The idea is to develop a web application which manage windows, doors, radiators and lights. We use OneM2M architecture to simulate sensors and actuators in order to retrieve their values through a Restful web services. According to the measurements, the service can trigger actuators to manage rooms.

Cloud computing

The application of the cloud part is during practical sessions. We discover basic functionalities that cloud services can provide. We study virtualization hosts, virtual machines and containers. We specifically use VirtualBox as virtualization tool, Docker as container tool and OpenStack as cloud service. We configure network topology and set up a web 2-tier application.

This course match very well with the Smart Device module. It gathers high-level competency of smart device. In this way it permits to understand how every IoT entities can communicate over a network to create a smart application, even if there are not the same type.
Regarding Middleware and Service and Smart Device module, I feel now confident with the complete design on an IoT systems. I have skills regarding standards, technologies and services available in order to choose the best solution according to specifications.
